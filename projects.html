Impedance Control in Dual Robots

This project involved connecting two KUKA LBR iiwa collaborative robots using ROS. The robots were controlled using impedance control and TF to minimize the distance between their end-effectors.

Patient Tracking in Surgical Navigation

This project involved tracking a patient's movements using a combination of different sensors, an optical tracking system, and a repurposed haptic device. The project also included the fusion of sensor measurements to improve tracking accuracy.

Motion Planning and Decision Making

This project implemented an end-to-end motion planning pipeline using RRT* as a global planner and Model Predictive Control as a local planner capable of velocity obstacle avoidance.

Pedestrian Detection Using Machine Perception

This project involved the detection of pedestrians using a combination of Lidar, Radar, and Image data on the TU Delft Database.

Drone Control Using Machine Learning

In this project, OpenAI gym and a labelled dataset were used to train an agent to control a drone based on pixel data from frames.
