About This Website

This website is hosted on GitHub Pages and built with HTML and JavaScript. Rendered on the landing page is a simple 2dof 2D canvas robot model. A mousetracker combined with P(ID) control is implemented to follow the position of the mouse and control the robot based on Jacobian calculations.
